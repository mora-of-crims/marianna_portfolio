## Стек

**Frontend:**
- HTML5, CSS3 (кастомный CSS без фреймворков), vanilla JavaScript
- Шрифты: Space Mono, Syne (Google Fonts)
- Кастомный курсор, анимации, адаптивная вёрстка

**Backend:**
- Node.js + Express
- Resend — отправка писем
- dotenv — конфигурация окружения

## Как запустить

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/marianna-portfolio.git
cd marianna-portfolio
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Создать `.env` файл

Скопировать `.env.example` и заполнить переменные:

```bash
cp .env.example .env
```

Содержимое `.env`:

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your@gmail.com
SMTP_PASS=your_app_password
OWNER_EMAIL=your@gmail.com
GEMINI_API_KEY=your_gemini_api_key
PORT=3000
```

> Для Gmail нужен [App Password](https://myaccount.google.com/apppasswords), не обычный пароль.

### 4. Запустить сервер

```bash
# Продакшн
npm start

# Разработка (с авто-перезапуском)
npm run dev
```

Сайт откроется на `http://localhost:3000`

## Форма обратной связи

Форма реализована на чистом JavaScript без сторонних библиотек.

**Валидация на фронте:**
- Имя — минимум 2 символа
- Телефон — регулярное выражение на допустимые форматы
- Email — проверка формата

**На бэкенде (`/api/contact`):**
- Повторная валидация всех полей (server-side)
- При успехе отправляются два письма параллельно через `Promise.all`:
  - Владельцу сайта — с данными заявки
  - Пользователю — подтверждение о получении
- Красивые HTML-письма с брендингом

## Что пришлось переписывать и доделывать вручную

Несмотря на активное использование ИИ, большая часть финального результата —

- Серверная валидация и обработка edge-кейсов — ИИ давал слишком упрощённые варианты
- Интеграция Gemini API: настройка, системный промпт, обработка ошибок
- Финальная структура проекта, роутинг, конфигурация окружения
- Тексты и подача — переписаны под свой голос и стиль
