# Зуева Марианна Сергеевна — Portfolio

Мой персональный сайт-портфолио. Лендинг с формой обратной связи.

---

## Важно

К сожалению, не получилось добавить AI-интеграцию по причине отсутствия платной подписки на LLM-сервисы. Копия письма пользователю также не отправляется — бесплатные SMTP-сервисы либо блокируются хостингом, либо ограничивают отправку на сторонние адреса. Вместо копии при успешной отправке формы показывается уведомление о том, что сообщение получено и я свяжусь в ближайшее время.

Поверьте на слово — я умею интегрировать что угодно и куда угодно. В рабочих проектах я не раз это делала, когда подписки оплачивал работодатель. В личных аккаунтах таких подписок нет.

## Стек

**Frontend:**
- HTML5, CSS3 (кастомный CSS без фреймворков), vanilla JavaScript
- Шрифты: Space Mono, Syne (Google Fonts)
- Кастомный курсор, анимации, адаптивная вёрстка

**Backend:**
- Node.js + Express
- Resend — отправка писем
- dotenv — конфигурация окружения

## Как запустить

### 1. Клонировать репозиторий

```bash
git clone https://github.com/mora-of-crims/marianna_portfolio.git
cd marianna_portfolio
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Создать `.env` файл

```bash
cp .env.example .env
```

Содержимое `.env`:

```
RESEND_API_KEY=your_resend_api_key
OWNER_EMAIL=your@email.com
PORT=3000
```

### 4. Запустить сервер

```bash
# Продакшн
npm start

# Разработка (с авто-перезапуском)
npm run dev
```

Сайт откроется на `http://localhost:3000`

## Форма обратной связи

Форма реализована на чистом JavaScript без сторонних библиотек.

**Валидация на фронте:**
- Имя — минимум 2 символа
- Телефон — регулярное выражение на допустимые форматы
- Email — проверка формата

**На бэкенде (`/api/contact`):**
- Повторная валидация всех полей (server-side)
- При успехе письмо с данными заявки отправляется владельцу через Resend
- Пользователю показывается уведомление об успешной отправке
- Красивые HTML-письма с брендингом
- Обработка ошибок: loading / success / error состояния
